Running

Open a DOS prompt to the %CLIPJ%/bin/ directory.

Execute start-clipj.cmd

The application will then start running in the background. You'll see the ClipJ icon in the system tray. You may need to edit which icons are always visible in the system tray.

Happy clipboarding!
